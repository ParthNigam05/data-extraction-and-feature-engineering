# Data Extraction and Feature Engineering

A Python-based data preprocessing project demonstrating data extraction, cleaning, and feature engineering techniques using country demographics and anime datasets.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## 📂 Files
- `DataExtraction.ipynb`
- `FeatureExtraction.ipynb`
- `Countries.csv`
- `anime.csv`

## 📌 Features
- Data cleaning
- Handling missing values
- CSV processing
- Text feature extraction
- Dataset preprocessing
